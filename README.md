# Click Gás 🚀

O **Click Gás** é uma solução completa para o setor de delivery, focada em automatizar o pedido de gás e água. 

## 🛠 Tecnologias Utilizadas

* **Front-end:** HTML5, CSS3 (Responsivo) e JavaScript moderno.
* **Lógica:** Manipulação de DOM e `LocalStorage` para persistência de dados.
* **Funcionalidades:** Sistema de Login, Dark Mode e validação de formulários.

## 🚀 Funcionalidades Principais

* **Interface Responsiva:** Adaptada para todos os dispositivos.
* **Controle de Acesso:** Fluxo de Login que personaliza a experiência.
* **Dark Mode:** Tema claro/escuro com persistência via `localStorage`.
* **Interatividade:** Popup de inatividade e promoções dinâmicas.

## 📂 Estrutura do Repositório

- `index.html`: Página principal.
- `login.html`: Interface de acesso.
- `produtos.html`: Catálogo de ofertas.
- `css/`: Estilização.
- `img/`: Assets.
